차트 페이지 접근 & 종목 전환 방식

검색 결과 / 관심종목에서 차트로 진입하고, 차트 내에서 종목을 전환하는 UX 패턴

A

종목 리스트 사이드 패널

차트 오른쪽 사이드바에 종목 리스트 탭을 추가하여 Financials/AI Fusion 위에 배치.
[검색결과] [관심종목] [최근본] 3탭으로 구성.
종목 클릭 시 차트가 즉시 전환. 검색 결과에서 Chart 버튼 → 차트 페이지로 이동 + 해당 종목이 사이드 리스트에 로드됨.

┌─────────────────────┬──────────────┐
│ │ [검색] [관심] [최근]│
│ CHART AREA │ ● Samsung 78,400│
│ │ ● SK Hynix 42,100│
│ │ ● ecoprobm ... │
│ ├──────────────┤
│ │ Financials │
│ │ AI Fusion │
├─────────────────────┴──────────────┤
│ RSI │ MACD │ Revenue │
└────────────────────────────────────┘
B

상단 종목 바 + 드롭다운

차트 상단 topbar에 종목 검색 + 드롭다운 추가.
현재 종목 옆에 검색 아이콘 → 클릭 시 드롭다운으로 관심종목/최근 종목/검색 표시.
사이드바 공간을 차지하지 않아 차트 영역이 넓음. 단, 목록을 한눈에 보기 어려움.

┌────────────────────────────────────┐
│ 005930 Samsung ▼ 78,400 [🔍검색] │
│ ┌──────────────┐ │
│ │ ★ 관심종목 │ (dropdown) │
│ │ ◷ 최근 본 │ │
│ │ ◎ 검색결과 │ │
│ └──────────────┘ │
├─────────────────────┬──────────────┤
│ │ Financials │
│ CHART AREA │ AI Fusion │
│ │ Sector Compare│
├─────────────────────┴──────────────┤
│ RSI │ MACD │ Revenue │
└────────────────────────────────────┘
C

왼쪽 접이식 패널 (트레이딩뷰 스타일)

차트 왼쪽에 접을 수 있는 슬림 패널.
기본 접힘 상태 → 아이콘만 표시 (★ 관심, 🔍 검색, ◷ 최근).
클릭 시 250px 폭으로 펼쳐짐. 오른쪽 Financials 사이드바는 그대로 유지.
트레이딩뷰/블룸버그 터미널과 유사한 패턴으로 파워유저에게 익숙.

┌──┬──────────────────┬──────────────┐
│★ │ │ Financials │
│🔍│ CHART AREA │ AI Fusion │
│◷ │ │ Sector Compare│
│ │ │ │
├──┴──────────────────┴──────────────┤
│ RSI │ MACD │ Revenue │
└────────────────────────────────────┘

💡 공통 사항 (어떤 옵션이든):

  • Search 결과의 "Chart" 버튼 → 차트 페이지로 이동 + 해당 종목 로드
  • Portfolio Holdings 행 클릭 → 차트 페이지로 이동
  • Dashboard Live Cases 행 클릭 → 차트 페이지로 이동
  • 관심종목(Watchlist) 기능 신규 추가 필요
