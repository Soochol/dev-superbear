추가할 요소

A

마켓플레이스 검색 조건 탭

Saved Searches 사이드바에 "My Searches | Marketplace" 탭을 추가. Marketplace 탭에서 다른 사용자의 인기 검색 조건을 브라우징하고 복제(fork) 가능. 사용 횟수, 결과 수, 공유자 정보 표시.

B

DSL 미리보기 패널

에이전트가 자연어를 DSL로 변환할 때, 생성된 DSL 표현식을 코드 형태로 보여주는 패널. 사용자가 직접 DSL을 수정할 수도 있음. "Agent → DSL 변환 과정의 투명성" 제공.

C

결과에서 바로 파이프라인 실행

현재 "Run Pipeline on All" 버튼 + 개별 "Pipeline" 버튼은 있음. 여기에 파이프라인 선택 드롭다운을 추가하여 어떤 파이프라인으로 실행할지 선택 가능.

D

검색 조건 편집 인라인 UI

Saved Searches의 각 항목에 편집/삭제/공유 버튼 추가. 클릭하면 해당 검색 조건의 프롬프트가 메인 입력창에 로드되어 수정 가능.
