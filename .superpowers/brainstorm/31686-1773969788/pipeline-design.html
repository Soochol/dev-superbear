추가할 요소

A

블록 편집 모달 (AgentBlockPrompt)

블록 클릭 시 편집 모달 표시. 스펙의 구조화된 필드 — name, objective, input_desc, tools (체크박스), output_format, constraints, examples. 자연어로 입력하면 AI가 구조화, 직접 필드를 채울 수도 있음.

B

파이프라인 목록/선택 헤더

현재 "Semi Value Strategy v2.1" 하드코딩 → 드롭다운으로 내 파이프라인 목록 표시. + New Pipeline 버튼. 각 파이프라인의 마지막 실행 시간, 케이스 수 표시.

C

실행 상태 UI (Execution Progress)

Execute 버튼 클릭 후 나타나는 실행 진행 오버레이. 각 stage별 상태 (대기/실행중/완료), 에이전트별 출력 프리뷰, 전체 진행률. 실행 완료 시 케이스 생성 알림 + 케이스로 이동 링크.

D

자연어 → 구조화 변환 패널

"AI Generate" 버튼 강화. 자연어로 전체 파이프라인을 설명하면 AI가 3 섹션(분석/모니터링/판단)을 자동 구성. 변환 결과를 미리보기로 표시하고 승인/수정 가능.
