보강 제안

새 페이지들(포트폴리오, 모니터링, 마켓플레이스)의 요약을 Dashboard에 추가

A

현재 유지 (변경 없음)

Dashboard는 이미 충분. 새 페이지(모니터링/포트폴리오/마켓플레이스)는 사이드바에서 직접 접근하면 됨. Dashboard를 가볍게 유지.

B

포트폴리오 요약 추가

stat-grid에 "Total P&L" (총 손익), "Portfolio Value" (포트폴리오 가치) 추가. 하단에 포트폴리오 미니 요약 카드 (보유 종목 수, 실현/미실현 손익) 추가.

C

3열 그리드로 확장

현재 2열(Cases + Alerts)을 3열로: Live Cases | Recent Alerts | Monitoring Status. 모니터링 상태 카드에 현재 동작 중인 cron 블록 수, 최근 실행 결과, 다음 실행 시간 표시.

D

풀 대시보드 (B + C 결합)

stat-grid에 포트폴리오 지표 추가 + 3열 하단 그리드 (Cases | Alerts | Monitoring). 전체 플랫폼의 한눈에 보기.
